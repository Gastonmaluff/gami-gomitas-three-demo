# Gami Gomitas Three.js Demo

Landing interactiva de una pagina para probar Three.js, React Three Fiber y Drei con un paquete 3D de gomitas animadas.

## Stack

- Vite
- React
- Three.js
- @react-three/fiber
- @react-three/drei

## Como correr

```bash
npm install
npm run dev
```

Luego abre la URL local que imprime Vite, normalmente:

```text
http://127.0.0.1:5173
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
```

## Interaccion

Pasa el cursor sobre el paquete para ver una reaccion suave. Haz click sobre el empaque para activar el bounce del paquete y el movimiento de las gomitas dentro.
